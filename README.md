#Carbon Pawprint

Carbon Pawprint is a short decsion based game to raise awarness of sustainable travel, 
the game features endangered animals as the protagonist, during the game play the user must choose
options such as mode of transport, shopping desination etc. Behind each choice lies a tally system which
counts the 'sustainability' of the users choice this system was derived from real values obtained from 
eniromental repots, global emission reports and many more, this logic allows for the decisions to be "real"
in terms of consequnces at the end of the game the user is given a sustainability score based on their choices.
This game was developed during Bangor Universities 24 hour hackathon sponsored by PromptLee. Our team was awarded
1st place for the project. The game is avaliable to demo on itch.io https://bryngame.itch.io/carbon-pawprint
